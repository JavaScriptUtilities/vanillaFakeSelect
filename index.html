Vanilla Fake Select

This script duplicates your select element to totally skin it.
No library dependency ✌️

How to use it

Download the latest version and add in your head section :

<link href="js/fake-select.css" rel="stylesheet" type="text/css" />
<script src="js/fake-select.min.js"></script>

Load demo skin or use it to create your own :

<link href="js/fake-select-demo.css" rel="stylesheet" type="text/css" />

Init the script at domready or later :

new vanillaFakeSelect(document.getElementById('test-select'));

Settings

You can customize the script by adding custom settings at init.
